# NOVA Café — نوا کافه

A premium Persian café-restaurant landing page built with pure HTML, CSS, and vanilla JavaScript.

**Live demo:** https://rojanalizadeh.github.io/novacafe-ui

## Features
- Full RTL Persian layout
- Cinematic hero with real photography
- Tabbed menu with 5 categories and realistic Tehran pricing
- Scroll reveal animations
- Sticky navbar with scroll effect
- Loading screen
- Photo gallery with hover effects
- Reservation form
- Fully mobile responsive
- Floating CTA button

## Built with
- HTML5
- CSS3 (custom properties, grid, flexbox, animations)
- Vanilla JavaScript (IntersectionObserver, scroll events)
- Vazirmatn — Persian web font
- Unsplash — photography placeholders

## What this demonstrates
- RTL (right-to-left) layout for Persian/Arabic markets
- Premium dark UI design
- CSS animations without any library
- Mobile-first responsive design
- Real-world café website structure
